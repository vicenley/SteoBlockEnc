# Stereographic Projection for Quantum Block Encoding

This repository explores the stereographic projection as a quantum encoding strategy for quantum block encoding, with a focus on quantum signal processing applications.

## Project Structure

```
SteoBlockEnc/
├── ms/              # Manuscript and paper materials
├── src/             # Source code
│   └── stereo_block_enc/  # Main package
├── tests/           # Unit tests
├── docs/            # Documentation
├── examples/        # Example scripts and demonstrations
├── data/            # Data files (gitignored)
├── notebooks/       # Jupyter notebooks for exploration
└── scripts/         # Utility scripts
```

## Setup

1. Create and activate the virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate  # On Linux/Mac
# venv\Scripts\activate  # On Windows
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Research Focus

- Stereographic projection as quantum encoding strategy
- Quantum block encoding
- Quantum signal processing within stereographic projection framework
